## 核心功能一览

**滚动长截图**
这是很多人从系统截图跳来的直接原因。CleanShot X 支持滚动截屏，实时预览，不用再拼图了。而且截完还能直接在截图上再编辑、标注。

**截图美化**
截图后可以一键加上彩色背景、圆角，让截图马上变得"好看"——不再是屏幕原始样子直接甩给别人。

**录制视频 / GIF**
支持录制屏幕内容和摄像头画面，录完可以直接导出 MP4 或 GIF。做教程、录 bug 复现视频都挺方便。

**OCR 文本提取**
截图里有文字想复制？点一下，识别出来，连格式都能还原。

**截图隐藏桌面图标**
截图时自动隐藏桌面上所有图标，截完再恢复。这个细节其实很实用，不用再专门清一下桌面。

**云端分享**
截图或录屏可以自动上传到云端，生成一条分享链接，发给同事直接点开看，不用传文件。
